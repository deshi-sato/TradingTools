# scripts/__init__.py 仕様書

## 概要
- `scripts` パッケージをPythonモジュールとして認識させるための空ファイル。現在は機能的なコードを含まない。

## 役割
- 旧来のPythonバージョン互換のために `scripts` ディレクトリをパッケージ化し、相対インポート（例: `from scripts.common_config import ...`）を可能にする。
- 実行時に副作用を与えないよう空のまま維持されている。

## 運用上の注意
- ファイルを削除すると一部のパッケージインポートが失敗する可能性があるため、空でも保持する。
- 共通の初期化コードを置きたくなった場合は、軽量で副作用の少ない処理のみに留めること。
